Glamified Hair & Beauty Website
Welcome to the official website project for Glamified Hair & Beauty, a modern beauty brand specializing in premium wigs, hair products, and hair-care education.
This website is designed to be clean, responsive, easy to navigate, and user-focused.
 Project Overview
This project is a multi-page static website built with HTML and CSS. The site showcases the brand, its products, and helpful resources for customers such as wig-care guides and contact support.
 Pages Included
1. Home (index.html)
•	Brand introduction
•	Navigation to core sections
•	Display-ready structure for featured products or promotions
2. Shop (shop.html)
•	Grid layout for wig products
•	Product images & descriptions
•	Placeholder items ready for future database/e-commerce integration
3. About Us (about.html)
•	Brand mission
•	Company values
•	Background story section
•	Space for future team images or founder information
4. Wig Care (wig-care.html)
•	Educational wig maintenance tips
•	Images paired with each care step
•	Optional section for additional tips
•	“Back to Home” navigation link
5. Contact (contact.html)
•	Brand contact information
•	Contact form with Name, Email, Message
•	Social media links linked with Font Awesome icons
 Design & UI
•	Fully structured, user-friendly layout
•	Desktop & mobile responsive layout (CSS-based)
•	Clean typography & spacing
•	Consistent brand styling applied across all pages

How to Use / Run the Website
1.	Download or clone the project folder.
2.	Open index.html in any web browser (Chrome, Edge, Firefox, Safari).
3.	Navigate through the site using the menu bar.
No server setup is required — it's fully static and runs locally.
